# TUPLE

튜플은 한번 만들면 수정할 수 없다.immutable.

()를 통해 만든다

```
a  = (1,2,3)
```

인덱스/범위 접근 모두 가능

```
a[1:3] // (2,3)
```

index,count등도 사용가능

remove,pop등은 당연히 불가능



튜플내부의 리스트의 값은 수정가능하다

````
t=([1,2,3],[4,5,6])
t[0][0]=0 //ok
t[0] = 1 //error
````



## 언패킹

```
tuple = (10,20)
x,y = tuple 
//x=10, y=20
x,y = 10,20 //이런식으로 변수선언도 가능하다 ㄷㄷ; 이것은 10,20을 튜플이라고 인식한 후 언패킹하는 것이다.
```



## 스왑핑

튜플과 언패킹 덕분에 파이썬은 멋진 스왑핑이 가능하다

```
x=10
y=20
x,y = y,x
//x=20 and y = 10
```
